## 🎯 About The Project

The **Termux Tools Masterclass** is a comprehensive, self-paced interactive  
learning platform designed to teach you everything about using Termux — the  
powerful Android terminal emulator — from the ground up.

This is not just a list of commands. It is a **fully interactive course**  
that runs entirely inside Termux, featuring:

- 📖 **Structured Lessons** with detailed explanations
- 🔧 **Try It Yourself** sections for hands-on practice
- 📝 **Quizzes** after every module to test your knowledge
- 💪 **Exercises** with real verification of your solutions
- 📊 **Progress Tracking** that saves your progress between sessions
- 🏆 **Certificate Generation** upon course completion
- 🎨 **Beautiful Terminal UI** with colors, borders, and visual aids

Whether you just installed Termux or you're looking to level up your skills,  
this course guides you every step of the way.

## 🚀 Quick Start

Get up and running in under 2 minutes:

```bash
# Step 1: Update Termux
pkg update && pkg upgrade -y

# Step 2: Download the script
curl -O https://raw.githubusercontent.com/your-username/termux-masterclass/main/termux_tools_masterclass.sh

# Step 3: Make it executable
chmod +x termux_tools_masterclass.sh

# Step 4: Start learning!
bash termux_tools_masterclass.sh
```

> 💡 **That's it!** The script handles everything else automatically.

│   ├── try_it_yourself()     - Free-form practice shell
│   └── run_exercise()        - Verified exercise system

├── 💾 Progress System
│   ├── save_progress()       - Writes to ~/.termux_lesson_progress
│   └── load_progress()       - Reads saved state
│
├── 📚 Modules 1-12
│   ├── module_01() ... module_12()
│   └── Each contains: Lessons + Try It + Quiz + Exercises
│
└── 🏆 Results System
    ├── show_final_results()  - Score report & grade
    └── Certificate generator - Text certificate file

## ❗ Troubleshooting

### Common Issues & Solutions

#### Script won't run
```bash
# Problem: Permission denied
./Class.sh: Permission denied

# Solution:
chmod +x Class.sh
bash Class.sh
```

#### Colors not displaying
```bash
# Problem: Garbled characters or missing colors
# Solution: Ensure you're in Termux (not another terminal)
echo $TERM
# Should show: xterm-256color

# If not, set it:
export TERM=xterm-256color
```

#### Progress not saving
```bash
# Problem: Progress resets every session
# Solution: Check home directory is writable
ls -la ~/.termux_lesson_progress

# Reset progress manually if corrupted
rm ~/.termux_lesson_progress
```

#### Package installation fails
```bash
# Problem: Unable to install packages during course
# Solution: Update package lists
pkg update

# Change mirror if download is slow
termux-change-repo

# Try again
pkg install <package-name> -y
```

#### Termux API commands not working
```bash
# Problem: termux-battery-status: command not found
# Solution 1: Install the termux-api package
pkg install termux-api -y

# Solution 2: Also install the Termux:API Android app
# Download from: https://f-droid.org/packages/com.termux.api/
```

#### Script runs too fast / display issues
```bash
# Problem: Text overflows or display is broken
# Solution: Increase terminal font size or scroll up

# Also check terminal width
tput cols
# Recommended: 65+ columns
```

#### Cannot execute exercises
```bash
# Problem: Commands in exercises fail with permission errors
# Solution: Make sure you're in the workspace
cd ~/termux_lesson_workspace

# Reset workspace
rm -rf ~/termux_lesson_workspace
mkdir -p ~/termux_lesson_workspace
```
